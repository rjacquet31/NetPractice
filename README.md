# NetPractice

> **42 cursus** · Networking · TCP/IP, subnetting and routing.

A hands-on networking project: a series of small broken network topologies to diagnose and fix by setting correct IP addresses, subnet masks and routes.

## Features

- Understanding IPv4 addressing and subnet masks
- Configuring interfaces, gateways and routing tables
- Solving 10 levels of increasing difficulty

## Usage

```bash
# Completed entirely in the browser-based simulator.
# Configuration answers are saved as the level .json / config files.
```

## Key concepts

TCP/IP fundamentals, subnetting, CIDR and routing.
